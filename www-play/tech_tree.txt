	planks + bricks + concrete -> large buildings
	planks + bricks + concrete + steel -> huge buildings


objects
-------------------------------------------------------------

we have the following first class objects:

	BuildingClass	- building type (e.g. tower block, police station)
	JobClass	- job type (e.g. builder, astronaut)
	ResidencyClass	- residency type (e.g. small house, penthouse suite)
	GoodsClass	- goods type (e.g. bricks, computer chips)
	IndustryClass	- industry type (e.g. carpentry, space weapons)

	Building	- city scoped building instance
	Job		- building scoped job instance
	Residency	- building scoped residency instance
	Goods		- city scoped goods instance
	Industry	- city scoped industry instance


buildings have the following properties:
	building class id
	position
	build start time
	build end time
	multiple job records
	multiple residency records


a job links a specific building with a specific job class. for instance, a hotel might provide two job records - one for maids and one for reception staff.

jobs have the following properties:
	building id
	job class id
	count of positions available
	count of positions filled in-city
	count of positions filled out-of-city

residency records link people to buildings
	building id
	resident class id
	count of available resident space
	count of used resident space
